1. Eerste stap dat je moet doen is deze command in jou terminal in php storm/ VS code typen. (copieer wat in geel is getyped)

composer require symfony/asset-mapper symfony/asset symfony/twig-pack

Dit gaat wel lang duren dus geef het tijd.


2.1 Eerst moet je ".env" map helemaal kopieeren
2.2 Tweede moet je een nieuwe bestand maken die ".env.local" heet
2.3 Derde moet je" .env" pasten in het ".env.local" map
2.4 Vierde met precies regel 28 in ".env.local" map kopieeren en plakken onder regel 28 en "#" verwijderen

(kijk foto voor voorbeeld)


2.5 Je moet van regel 29 "series" veranderen naar jou eigen database naam
2.6 Bij regel 29 moet je alleen "MariaDB" hebben bij einde van alinea

3. Derde moet in het backup file kijken toets om het database te importeren (het staan in /backups en heet series.sql)
